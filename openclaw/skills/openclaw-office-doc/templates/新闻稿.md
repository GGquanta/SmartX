# xx科技发布企业版 v18.8.0：把"加 AI"变成"先加流程，再让 AI 顺着流程跑"

**青岛，2026年4月27日** — xx 科技有限公司（"xx科技"）今日正式发布 **企业套件企业版 v18.8.0**，以"先加流程、再让 AI 顺着流程跑"为核心设计理念，把企业 ERP / CRM / 项目管理 / 客户联系四个模块以一套 AI Agent 串联，覆盖从客户进线到项目交付的完整链路。

公司创始人表示："过去两年，市场上多数企业 AI 产品停留在'会聊天'层面，但真实业务里，AI 必须知道下一步该做什么、不该做什么。v18.8.0 把流程从隐性变成显性，让 AI 在公司流程里有可执行的脚本，而不是临时被人指挥。"

## 三大新特性

**一、流程驱动的 AI Agent**：每个业务流程节点显式定义"输入 / 输出 / 责任人 / SLA"，AI Agent 在节点内可调用工具、写文档、推送消息，但必须在节点边界做交付与审批，避免"擅自越权"。

**二、数字员工助手**：内置数字员工作为公司一号助手，对接企业微信、Odoo、Anthropic API、SMS 短信、Slack，可在 30 秒内完成"客户报价 + 合同初稿 + 内部周知"全流程。

**三、企业级安全闸门**：v2.8.1 落地的"三层闸门"机制（工具脚本 + AGENTS.md + SDK），LLM 输出的目标对象与 URL 必须经过 sanitizer，所有"@all 群发""跨群转发"必须二次确认。

## 客户验证

企业套件已在 50+ 中小企业完成部署。试用客户、青岛某外贸公司 CTO 反馈："以前我们的项目交付流程靠飞书文档维护，新员工三个月才能上手；用了企业套件之后，AI Agent 直接按流程问下一步怎么做，新人第一周就能独立处理普通工单。"
